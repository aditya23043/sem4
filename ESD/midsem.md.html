  3 ## Embedded System
  4 
  5 - Performs Specific tasks unlike general purpose computers within a larger system
  6 - Low power consumption
  7 - Generally, remotely deployed
  8 - Battery powered
  9 - Tightly integrated

 13 - Might not have a user interface

 17 ## Neumann vs Harvard Architecture
 18 
 19 | Neumann                                                    | Harvard                                        |
 20 | ---------------------------------------------------------- | ---------------------------------------------- |
 21 | Single bus for memory for both instruction and data        | Separate data busses for instruction and data  |
 22 | Sequential access to memory space for instruction and data | Parallel Access to instruction and data memory |

152 ### Open Drain
153 
154 - Remove PMOS
155 - For doing basic circuitry, like driving an LED, we don't necessarily need a dedicated PMOS
156 - We can connect the anode of the LED to 3V3 and cathode to the gpio pin (with only NMOS), which is internally grounded
157 - To turn on the LED, switch on the NMOS and close if you want to switch it off (no current will flow because open ckt)
158 - Cost effective &#x2713;
159 - pullup resistor required because with the GPIO can only pull to low using the NMOS
160 
161 > True open drain circuit -> no pullup resistor
162 
163 ## Schmitt Trigger
164 
165 - Detects if voltage is high or low
166 - can be considered as a 1 bit ADC
167 - prevents noise in input signal
168 - circuit protection
169 
170 ## ADC
171 
172 - Input, not through a schmitt trigger
173 - V_IL = Lowest voltage to get a digital HIGH ~0.3 x V_DD
174 - V_IH = Highest voltage to get a digital LOW
175 - I_LKG = leakage current
176   - when input connected to V src
177   - when NMOS and PMOS are off
178   - not needed
179   - goes into opamp (~infinite resistance)
180 - GPIO pins can only take in limited current ~ +- 6ma
181 - Source Current - Output (through PMOS)
182 - Sink Current - Input (through NMOS)
183 - SAR ADC - Successive Approximation Resistance ADC
184   - Cheap (num of gates)
185   - Fast
186   - Reliable
187 - LDR - Light dependent resistor
188 - Thermistor - resistor charges with change in temperature
189
